# Yamaha Style Assembler

A Python application built with CustomTkinter to assemble, merge, and edit Yamaha arranger style files (.sty) by replacing and mapping CASM sections.

## Features
- Extract and inject CASM chunks properly.
- Map markers (e.g. `Main A>Main B`).
- Automatic bar length sync with source styles.
- CustomTkinter dark mode GUI.

## Installation
``bash
pip install -r requirements.txt
``

## Usage
``bash
python main.py
``
